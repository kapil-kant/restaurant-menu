# Restaurant Menu

A simple, elegant static restaurant menu page built with HTML and CSS.

## Features

- 🍝 **5 Menu Categories**: Appetizers, Main Courses, Pasta, Pizza, Desserts
- 📱 **Fully Responsive**: Looks great on all devices
- 🎨 **Elegant Design**: Italian-themed color palette
- 🏷️ **Dietary Tags**: Vegetarian, Vegan, Gluten-Free, and Spicy indicators
- 🔗 **Smooth Navigation**: Sticky navigation with anchor links

## Tech Stack

- HTML5
- CSS3
- Google Fonts (Playfair Display, Lato)
